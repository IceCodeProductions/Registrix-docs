# Customization Settings

This step only needs to occur once or when you decide to change the operational mode of Registrix.

## Menu Location

From the left side menu, click on **Customization**, then click **Settings**.

## Procedure

Select the **Domain Mode** radio button.  Then click the **Update** button.  Be sure the Viewer Login method is set to Per Site.

The Registrix service is now in the Domain mode.
